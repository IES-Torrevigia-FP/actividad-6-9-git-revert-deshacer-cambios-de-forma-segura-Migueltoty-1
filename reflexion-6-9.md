##La diferencia
git reset es como “borrar” commits y volver atrás en el tiempo.
git revert no borra nada, hace un commit nuevo que deshace otro.

##Cuando usar git revert
Cuando ya subiste los cambios y alguien más puede haberlos usado.
En ramas compartidas donde no quieres liarla cambiando el historial.

##Forward-moving undo
Porque no va hacia atrás, sino que sigue adelante creando otro commit que cancela el anterior.
